## Movie App TEST

#Prequirement

- php version 7.2
- mysql

#Installation
- install apache2
- composer install or composer update
- php artisan serve migrate
- php artisan serve db:seed (initialize database and pull movie data from imdb)
- npm install 
- npm run prod (this is just mix build for resource)
